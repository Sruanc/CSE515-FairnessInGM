# Extension: Fairness Analysis of Crash Reporting Completeness

Applies FVGM fairness principles to Washington State crash data, analyzing whether reporting completeness exhibits bias across Vulnerable Road User (VRU) groups.

## Notebooks (Run in Order)

1. **01_VRU_Data_Preparation.ipynb** - Load raw data, create features and target variable
2. **02_VRU_Collider_Bias_Demo.ipynb** - Demonstrate collider bias effect in observational data  
3. **03_VRU_Decision_Tree_Training.ipynb** - Train decision tree classifier
4. **04_VRU_FVGM_Verification.ipynb** - Compute fairness metrics using FVGM logic
5. **05_VRU_Stratified_Analysis.ipynb** - Analyze fairness across demographic groups

## Methodology

This extension applies **FVGM-inspired empirical fairness analysis** to real-world crash data:

- **Formal FVGM reproduction**: Implemented in `justicia/` folder (benchmarks)
- **FVGM-inspired extension**: Applied principles to novel domain (this folder)
- **Key insight**: Group Data Quality may be a fairness-relevant factor

### Two-Part Project Structure

1. **Part 1: Reproduction** - Formal FVGM verification on Adult/COMPAS benchmarks
2. **Part 2: Extension** - Empirical proof-of-concept on VRU crash data
